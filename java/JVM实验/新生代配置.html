-XX:NewRatio=2指定了老年代与新生代之比为2

总结

  • 优化策略：尽可能将对象留在新生代，减少老年代GC的次数，因为老年代GC比较耗时
  • -XX:ServivorRatio=eden/from=eden/to，用于配置eden和servivor的比例
  • -XX:NewRatio=老年代/新生代，用于配置老年代与新生代的比例
