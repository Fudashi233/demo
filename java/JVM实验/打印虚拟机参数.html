打印虚拟机参数

源码

package test;

import java.lang.management.ManagementFactory;

public class Test {

    public static void main(String[] args) {
        printCommand(); //打印虚拟机参数
    }

    /**
     * 打印虚拟机配置参数
     */
    private static void printCommand() {
        System.out.printf("VM arguments:%s\n",ManagementFactory.getRuntimeMXBean().getInputArguments());
    }
}

VM arguments

-Xmx20M -Xms1M -XX:+PrintCommandLineFlags

console

-XX:InitialHeapSize=1048576 -XX:MaxHeapSize=20971520 -XX:+PrintCommandLineFlags -XX:+UseCompressedClassPointers -XX:+UseCompressedOops -XX:-UseLargePagesIndividualAllocation -XX:+UseParallelGC

VM arguments:[-Xmx20M, -Xms1M, -XX:+PrintCommandLineFlags, -Dfile.encoding=UTF-8]

总结

  1. ManagementFactory.getRuntimeMXBean().getInputArguments()会打印传递给虚拟机的参数

  2. -XX:+PrintCommandLineFlags可以打印传递给虚拟机的显式和隐式参数

  3. -XX:+PrintVMOptions打印传递给虚拟机的显式参数

  4. -XX:+PrintFlagsInitial打印虚拟机的各项初始参数

  5. -XX:+PrintFlagsFinal打印虚拟机的各项参数
